Angular 20 Student Management System (scaffold)
---------------------------------------------

This is a minimal Angular 20-compatible project scaffold with the Student Management app.

To use:
1. Unzip into a folder.
2. Run `npm install` to install dependencies.
3. Run `ng serve` to start the dev server.

If you have issues with Angular CLI versions, run:
  npm i -g @angular/cli@^20.0.0
